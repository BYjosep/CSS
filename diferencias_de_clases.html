Diferencias de classes

La principal diferencia que existe en el id y class es que que el Id debe ser utilizado una sola vez dentro de la página en la que se está trabajando, mientras que la clase se puede repetir varias veces para diferentes elementos.

EL id debera ser usado para identificado la parte principal dentro de el mismo archivo. Y class debera ser usado para lineas que se van a repetir en ese rama principal.

El id en un conjunto de paragrafos donde se hace un resumen de un libro, en el primer paragrafo(Este podria ser un titulo h2 con resumen de "nombre de libro") pondras el id y en paragrafos donde quieras separar por ejemplo los temas y ponerlos de color azul deberas de ponerlos en un class.

El id es representadoen CSS con . (esto se represemtaria .estoEsUnId) y class es identificado con #(esto se represemtaria #estoEsUnaClass)
